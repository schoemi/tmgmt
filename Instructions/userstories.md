# User Stories

## Gigs

Als User möchte ich Gigs Datensätze für Termine an denen meine Band spielt anlegen.

Als User möchte ich Gig Datensätze verändern können.

Als User möchte ich Gig Datensätze lösche können.

Als user möchte ich den Status eines Gigs ändern können.

Als User möchte ich Aktionen zu einem Gig ausführen können.

## Aktionen

Als User kann ich Aktionen an einem Event ausführen um Daten zu erfassen oder Automatisierungen zu starten.

Als User kann ich eine Aktion anlegen und auswählen ob diese Aktion eine Dokumentation von Daten erzeugt oder ob ein Webhook aufgerufen wird.

Als User kann ich eine Aktion bearbeiten oder löschen.

Als User kann ich einen Webhook anlegen und diesen mit Namen, Beschreibung HTTP Methode, URI versehen.


Als user kann ich unterschiedliche Dokumentations-Aktionen anlegen um einzelne Felder in meinem CPT zu befüllen oder Log-Einträge zu verfassen.
